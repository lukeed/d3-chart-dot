lukeed / d3-chart-dot Dot chart with D3.js

Install

npm install --save d3-chart-dot

Usage

The chart module accepts one parameter, an options object.

var DotChart = require('d3-chart-dot');

var chart = new DotChart(options);

chart.render([{x, y}, {x, y}, {x, y}]);

Options

Some generic comment about all the options before going into specifics.

target

Type: string
Default: '#chart'
The element selector.

width

Type: integer
Default: 500
The maximum width of the chart area.

height

Type: integer
Default: 80
The maximum height of the chart area.
